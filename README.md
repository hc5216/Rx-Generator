# Rx Generator

> **在线演示：** [https://rxg.20110208.xyz](https://rxg.20110208.xyz)

一个用于处方笺版式编辑与导出的静态网页工具。

> 本项目仅用于网页排版、UI 设计和文档样式演示学习，不具备真实医疗文书或诊疗服务效力。

## 功能

- 直接编辑医院名称、患者信息、诊断、药品、签名和审核信息
- 支持两种原创通用视觉预设：经典处方、极简账册
- 支持独立选择签名字体：标准楷体、马善政、龙藏、知芒行、刘建毛草
- 签名字体和版式预设均不会清除当前填写内容，并会记住最后一次选择
- 支持导出当前处方内容为：JPG、PNG、PDF、SVG
- 导出时仅包含处方笺区域，不包含页面背景、免责声明或导出工具栏
- PDF 会自动选择适合的横竖方向并缩放至单页 A4

## 使用方法

1. 使用浏览器打开 `index.html`。
2. 阅读并关闭免责声明。
3. 点击或输入以编辑处方内容。
4. 在“样式预设”中选择需要的视觉样式。
5. 在“签名字体”中为签名文本选择展示字体。
6. 点击页面右上方的“导出”，选择所需格式下载文件。

## 技术栈

- HTML
- CSS

## 字体许可

马善政、龙藏、知芒行、刘建毛草均为 [Google Fonts](https://fonts.google.com/) 提供的字体，并以 **SIL Open Font License 1.1** 发布。项目在 `assets/fonts/` 中自托管字体文件及各字体对应的 `OFL.txt` 许可证文本。

## 注意事项

- 样式预设为原创的通用视觉设计，仅用于排版演示；不复刻或对应任何真实医疗机构、官方处方或正式医疗文书。
- 签名字体仅改变文本的艺术展示效果，不生成、仿制或验证真实手写签名、电子签章，也不具备任何法律或医疗文书效力。
- 图片、PDF 和 SVG 导出依赖 CDN 加载的第三方库，请确保使用时网络连接正常。JPG、PNG、PDF 会保留导出时的字体外观；SVG 脱离本网站打开时，如系统未提供对应字体，可能显示为回退字体。
